# Lithuanian Alphabet Game

A small browser game for practicing the Lithuanian alphabet order.

Open `index.html` in a browser and choose which letter fits the blank space.

Levels:

- Level 1: choose the next letter after a short sequence.
- Level 2: see one letter and tap the next three letters one by one.
- Level 3: see a five-letter sequence with one visible clue and fill the missing letters one by one.

Alphabet used:

`A Ą B C Č D E Ę Ė F G H I Į Y J K L M N O P R S Š T U Ų Ū V Z Ž`

The Lithuanian alphabet has 32 letters and does not include Q, W, or X.
